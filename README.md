# 🎉 Gender Reveal App

Нежный, волшебный одностраничный сайт для объявления пола будущего малыша.

## 🚀 Быстрый старт

```bash
# Установить зависимости
npm install

# Запустить dev-сервер
npm run dev

# Собрать для продакшена
npm run build
```

## 🔧 Сменить пол ребёнка

Открой файл `src/stores/gender.ts` и измени строку:

```ts
// Смени здесь: 'girl' | 'boy'
const gender = ref<Gender>('girl')
```

## ✨ Возможности

- **Конфетти/салют** — при загрузке и по нажатию
- **Параллакс** — гироскоп телефона (DeviceOrientation API) + fallback мышь/тач
- **Плавающие эмодзи** — можно перетаскивать пальцем
- **Переключатель пола** — кнопка 🩷/💙 (для демо)
- **iOS-гироскоп** — кнопка «📱 Гироскоп» запрашивает разрешение
- **PWA-ready** — можно добавить на главный экран телефона
- **Адаптив** — оптимизировано для 360–428px (iOS + Android)

## 📁 Структура

```
src/
├── components/
│   ├── App.vue              # Корневой компонент
│   ├── BackgroundLayer.vue  # Анимированный фон (параллакс)
│   ├── GenderReveal.vue     # Главный блок с надписью
│   ├── FloatingEmojis.vue   # Интерактивные плавающие эмодзи
│   └── Controls.vue         # Кнопки управления
├── composables/
│   ├── useParallax.ts       # Гироскоп + mouse fallback
│   └── useFireworks.ts      # canvas-confetti обёртка
├── stores/
│   └── gender.ts            # Pinia store (пол, цвета, эмодзи)
└── assets/
    └── main.css             # Глобальные стили
```

## 📱 Для iOS (гироскоп)

На iPhone требуется жест пользователя для запроса разрешения.
Кнопка «📱 Гироскоп» появится автоматически — нажми, разреши.

## 🌐 Деплой

```bash
npm run build
# Папка dist/ — готова к загрузке на любой хостинг (Netlify, Vercel, GitHub Pages)
```
